# User Service

## Description
The User Service manages user identities, profiles, and authentication within the Virtual Office ecosystem.

## Key Features
- User Registration and Login
- Profile Management
- Authentication & Authorization (Security)

## Configuration
- **Port**: `8081`
- **Database**: MySQL `virtual_office`

## Setup
1. Ensure the database is running.
2. Run the service: `mvn spring-boot:run`
