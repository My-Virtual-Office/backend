# Shared Library

## Description
This module contains shared utilities, DTOs, constants, and configuration classes used across multiple microservices in the Virtual Office backend.

## Usage
This library is included as a dependency in other services.

```xml
<dependency>
    <groupId>com.virtualoffice</groupId>
    <artifactId>shared-library</artifactId>
    <version>${project.version}</version>
</dependency>
```

## Contents
- **DTOs**: Common Data Transfer Objects.
- **Utils**: Helper classes.
- **Exceptions**: Global exception handling.
